# CalculadoraJava

[Vídeo Tutorial Base | Kenny Yip Coding](https://youtu.be/jQo6n-i6wpo?si=YpuVRe8b9S5_pLOJ)

## Sobre

Calculadora feita em Java(Swing/AWT) que simula uma calculadora similar as dos dispositivos Apple. Pretendo implementar mais opções, como deixar o botão de raiz quadrada funcionando e opções de cosseno, seno e tangente.
